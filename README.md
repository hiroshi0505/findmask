# アプリケーション名
NumberApp
# アプリケーション概要
今は、小学生でもパソコンを使う時代です。そんな子どもたちが簡単に数字の学習を行うことのできるアプリケーション。<br>
また、外国には日本には無い数字に関するジェスチャーが存在します。そのようなジェスチャーも覚えていたらいつか役立つかも。。。<br>
このアプリケーションでは、そのような数字にまつわる学習を行えます。
# 説明
このアプリケーションでは数字にまつわる学習を行えます。まず、「学習ページ」では３つのカテゴリを用意しました。一番左のカテゴリについて説明します。ここには日本語の漢数字の画像を「一」から「十」まで順番に並べました。左右にスクロールすることで各画像を確認することが可能です。また、数字のボタンをクリックすることで目的の画像に飛ぶことができます。例えば「10」のボタンをクリックすると「十」の画像を確認できます。<br>ユーザーは自由に投稿が可能で、投稿一覧ページにてユーザーの投稿を閲覧できます。詳細ボタンをクリックすると詳細ページに遷移し、コメントの入力も可能です。投稿した本人であれば編集と削除が可能です。投稿者の名前をクリックするとその投稿者のページに遷移し、その投稿者の投稿だけを閲覧できます。<br>検索バーに検索したい内容を入力すると、ヒットした内容の投稿だけを閲覧できます。

# 利用方法
まずアカウント登録が必要ですので「新規登録」をクリックします。新規登録画面に全て正しく入力し、「新規登録」をクリックすると、アカウントが作られログインします。ログアウトするとログイン画面に戻ります。ログイン画面では、登録したメールアドレスとパスワードを入力し「ログイン」をクリックするとログイン状態となり、アプリケーションを使用できます。<br>投稿したい場合は「投稿する」をクリックすると、投稿画面に遷移しますので画像を選択し、説明文を入力した後、「SEND」をクリックすると、投稿が完了します。投稿した内容は投稿一覧ページに表示されます。<br>「詳細」をクリックすると詳細ページに遷移します。詳細ページではその投稿に対して、コメントを投稿することが可能です。コメントを入力し「SEND」をクリックするとコメントの投稿が完了し＜コメント一覧＞に表示されます。<br>投稿者本人であれば「編集」と「削除」が可能です。編集したい場合は「編集」をクリックします。すると編集ページに遷移しますので画像と説明文を変更できます。変更した後、「SEND」をクリックすると投稿内容が上書きされます。削除したい場合は「削除」をクリックするとその投稿が削除されます。<br>投稿について検索したい場合は、検索バーに内容を入力し、検索をクリックします。するとヒットした内容の投稿が一覧で表示されます。
# 目指した課題解決
例えば、日本語の漢数字を画像だけ見て覚えたい方や、外国での数字の表し方を画像を見るだけで学習したい方に使用してもらうことで、簡単に学習ができます。
